Enumeration Type Documentation

enum E_HDLCD_PACKET

The enum E_HDLCD_PACKET to specify all possible HDLCd packet types.

This enum specifies the set of packet types of the HDLCd access protocol. Each of them is represented by a specific derived class of base class HdlcdPacket.

Enumerator
HDLCD_PACKET_DATA 

A data packet of the HDLCd access protocol.

HDLCD_PACKET_CTRL 

A control packet of the HDLCd access protocol.

HDLCD_PACKET_UNKNOWN 

Unknown packet type.
